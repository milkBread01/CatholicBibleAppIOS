This App is a swift application to create a Catholic Companion app which will contain the following from public domain sources such as https://gutenberg.org/ and https://librivox.org/:
★★★ Bible Text ★★★
▶︎ Douay-Rheims Bible Text with notes
  ▷ User will be able to add highlights (multi-color) and notes as well as bookmark pages and verses.

★★★ Library/Resources ★★★
▶︎ Baltimore Catechism (1-3)
▶︎ Summa Theologica
▶︎ Optional Librevox downloads
▶︎ Find Local Parish 
▶︎ Liturgical Calendar
▶︎ Weekely Mass
▶︎ Interactive Rosary

★★★ Settings ★★★
▶︎ Change Theme, font size
▶︎ Account Status
▶︎ Allow App to use current location (to use parish lookup)
▶︎ Allow Push notifications
▶︎ Set daily reminder (push notification) to remind user to read x amount of time or n number of chapters
